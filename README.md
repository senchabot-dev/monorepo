## Getting Started
### Prerequisites
* Node 18 or later for Reflect, Interface & Discord bot
* Go 1.19 or later for Twitch bot

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/senchabot-dev/monorepo.git
   cd monorepo
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run turbo build
   ```sh
   npx turbo build
   ```
